🚀 Automação de Envio de Solicitações com VBA
📌 Descrição

Projeto desenvolvido em VBA para automação de processos administrativos, com foco na organização de dados e geração automática de e-mails a partir de informações estruturadas em Excel.

⚙️ Funcionalidades

Validação de campos obrigatórios

Registro automático de dados em base estruturada

Organização e tratamento de informações operacionais

Geração automática de e-mail com dados dinâmicos

Integração com Outlook Web via link parametrizado

Limpeza automática de campos após execução

🧠 Tecnologias Utilizadas

VBA (Visual Basic for Applications)

Microsoft Excel

Integração com Outlook Web

📊 Objetivo do Projeto

Automatizar rotinas manuais, reduzindo erros operacionais e aumentando a eficiência no envio de solicitações internas.

🔒 Observação

Este projeto é uma versão demonstrativa baseada em um processo real de ambiente corporativo.
Todos os dados, e-mails e informações foram anonimizados para fins de estudo e portfólio.

💡 Aprendizados

Estruturação de código em VBA

Automação de processos empresariais

Manipulação e persistência de dados

Integração entre sistemas

Boas práticas de organização e modularização

🚀 Possíveis melhorias futuras
